# Project 31: Currency Converter 💱

A Java CLI tool to convert currencies using fixed exchange rates.

## 💱 Supported Rates

- INR → USD: 0.012
- USD → INR: 83.0
- INR → EUR: 0.011
- EUR → INR: 90.0
- USD → EUR: 0.93
- EUR → USD: 1.08

## ▶️ How to Run

```bash
mvn compile
java -cp target/classes com.example.CurrencyConverter
